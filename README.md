# SalesForce360  Regional Sales Executive Performance Dashboard
**Project Description:**

This project is an interactive Excel-based Sales Performance Dashboard that analyzes and compares sales executives' performance across four regional zones — Central, North, South, and West. It tracks total sales revenue, target achievement rates, and deviation from targets giving managers a powerful tool to evaluate team performance and make data-driven decisions.

**Dataset Description**

The dataset contains regional sales records with the following key fields:


Sale Executive — Name of the sales representative

Sum of Total Sales — Total revenue generated (in PKR/INR)

Count of Target Hit% — Percentage of targets successfully achieved

Count of Away from Target% — Percentage deviation from assigned targets

Region — Geographic zone (Central, North, South, West)

Dashboard Filter — Interactive buttons (Dashboard 1, 2, 3, 4)
